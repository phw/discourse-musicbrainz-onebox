# Discourse MusicBrainz Onebox plugin

This plugin adds Onebox support for MusicBrainz.org. Use it to quickly display
information about artists, releases, labels and more.

## Supported entities

- [x] Artists
- [x] Release Groups
- [x] Releases
- [ ] Recordings
- [x] Works
- [x] Labels
- [ ] Place
- [ ] Series
- [ ] Events
